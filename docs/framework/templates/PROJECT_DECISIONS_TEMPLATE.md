# Project Decisions

## Purpose

This document records important decisions made during the project.

- Prevent re-discussion
- Provide context for future changes
- Help AI understand constraints

---

## Decision Format

Each decision must follow this structure:

### [Decision Title]

**Context**  
Why this decision was needed.

**Decision**  
What was decided.

**Alternatives Considered**  
Optional — what else was considered.

**Impact**  
What this affects.

---

## Decisions

### Example Decision

**Context**  
Need to handle form validation.

**Decision**  
Use react-hook-form + zod.

**Impact**  
All forms must follow this pattern.

---

## Rules

- Record ONLY meaningful decisions
- Do NOT log trivial changes
- Update when decisions change
- Keep concise
